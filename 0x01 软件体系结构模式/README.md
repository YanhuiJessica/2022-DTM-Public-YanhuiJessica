# Java 开发环境配置

- 安装 JDK（Java Development Kit）

    ```bash
    sudo apt install default-jdk
    ```

- 安装 Eclipse

    ```bash
    sudo snap install eclipse --classic
    ```
- 安装 VS Code [Java 扩展](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack)
  - 当前配置不能很好地应对缺失导入包的情况
